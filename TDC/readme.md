In pure behavioral Verilog, simulating sub-gate propagation delays accurately is difficult. To mathematically model an effective measurement resolution of 10ps in the testbench, this design utilizes a simulated Vernier Delay Line using transport delays (#0.020 and #0.010). This allows the system-level digital logic to be verified against ultra-fine timing intervals.
However translating this dual line architecture into physical silicon would be inefficent as it requires routing perfectly matched delay lines through general logic which will consume a lot of area and is easily sucseptible to pvt variations which can drift due to sudden temperature echanges and just increases complexity simply.
for fully synthesizable code and thus physical implementation we will use carry4 moving forward to simplify the routing complexity and make it less affected by temperature variations.But for now we have used this to show mathematically how we achievee a 10ps resolution tdc.
